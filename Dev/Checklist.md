# Addon Development Checklist

This document tracks missing features from the old addon that are left to add.

## Missing Player Management Panel Features

- [x] **View Detailed Flags:** Add a button to show a player's specific anti-cheat violation counts.
- [x] **Reset Player Flags:** Add a button to reset a player's flags to zero.
- [x] **Toggle Watch:** Add a button to add/remove a player from a "watch list" for easier monitoring.

## Other Missing Commands

- [x] `myflags`: A command for players to check their own flag count.
- [x] `listranks`: A command to list all configured ranks.
- [x] `listwatched`: A command to list all players on the watch list.
- [x] `purgeflags`: A command to wipe a player's flag data.
- [x] `worldborder`: A command/system to manage the world border.
